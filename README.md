# functionbook

functionbook, a facebook clone, is a social media website that allows users to share snippets of their life with people they care.
___

## Tools Utilized

* Ruby on Rails
* JavaScript
* jBuilder
* PostgreSQL
* React
* Redux
* Amazon WEb Services S3

## Demo

[functionbook live](functionbook.herokuapp.com)
